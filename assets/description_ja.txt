# app name

Monotweety - 通知エリアからツイート

# short description

端末の通知エリアからツイートすることに特化したTwitterクライアント

# description

Monotweety（モノツイーティ）は通知エリアからツイートすることに特化したシンプルなTwitterクライアントです。

お使いの端末がAndroid 7.0以上であれば、Android 7.0からの新機能を利用して、文字通り「通知エリアから」ツイートすることができます。
それ以前の端末では、通知をタップすると編集ダイアログが立ち上がります。

また、編集ダイアログからはツイートを前のツイートへのリプライとして繋げる「連ツイ」をワンタップで有効にできます。

機能一覧
- 通知エリアからツイート
- 連ツイ(複数のツイートをリプライとして紐付け)
